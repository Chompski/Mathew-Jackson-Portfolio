Im Mathew Jackson, a Front-end developer who is creative, passionate and eager to learn.

Antdentifier is a personal full-stack project I created, it is an application for hobbyist ant keepers, the main goal for the application is to help users identify and care for a multitude of ant species. The application achieves this by using a process of elimination, users can input their own criteria and break down the database on the backend to find the closest match for the criteria given.
